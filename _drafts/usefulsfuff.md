## Fairlearn
[Fairlearn](https://fairlearn.org/){:target="_blank"} is an open-source, community-driven project and Python library designed to help data scientists and developers assess and improve the fairness of artificial intelligence systems. The toolkit focuses on identifying and mitigating "harms" to specific groups of people—such as those defined by race, gender, or age—by providing specialized metrics for evaluation and algorithms for bias mitigation. Beyond its technical tools, the project emphasizes that fairness is a "sociotechnical" challenge, offering educational resources and case studies to help practitioners navigate the complex trade-offs between model performance and equity.

## Ragas
[Ragas](https://www.ragas.io/) is an open-source Python framework designed to help developers move from subjective "vibe checks" to systematic, metrics-driven evaluation for Retrieval-Augmented Generation (RAG) and LLM applications. It provides a suite of specialized metrics—such as faithfulness, answer relevance, and context precision—to quantify the performance of both the retrieval and generation components of an AI pipeline. Beyond just scoring, Ragas features automated synthetic test data generation to create diverse evaluation datasets and integrates seamlessly with popular tools like LangChain and LlamaIndex. By using LLMs as judges to provide objective, reference-free scores, it enables teams to iterate quickly, detect regressions, and ensure the reliability of their AI systems in production.
