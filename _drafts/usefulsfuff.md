## Fairlearn
[Fairlearn](https://fairlearn.org/){:target="_blank"} is an open-source Python toolkit that enables data scientists to evaluate and mitigate AI-driven harms through specialized fairness metrics and bias-reduction algorithms.

## Ragas
[Ragas](https://www.ragas.io/) is an open-source Python framework that provides a suite of metrics for evaluating Retrieval-Augmented Generation (RAG) applications by quantifying the accuracy of both retrieved context and generated answers.
